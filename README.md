# タスク管理アプリ

考えずに行動に移すための、シンプルなタスク管理アプリです。
タスクを「思いつき / 今日やる / 明日やる」に分けることで、
**やることを溜めすぎず、今やるべきことに集中できる設計**にしています。

※ 初回起動時は、サーバーの起動により表示までに数秒かかる場合があります。

---

## デモ

[https://todo-app-liard-ten.vercel.app/](https://todo-app-liard-ten.vercel.app/)

## 主な機能

* タスクの追加 / 編集 / 削除 / 復元
* タスクの分類（思いつき / 今日やる / 明日やる）
* ドラッグ＆ドロップによるタスク移動
* タスクの並び順の保存（DB連携）
* 日付切り替え時に「明日やる」タスクを「今日やる」に自動移動
* 完了したタスク・ごみ箱の管理
* フィルター切り替え（全て / 完了 / 削除）
* データベースへの永続化（PostgreSQL + Prisma）
* ローディング時のSkeleton UI表示

---

## 制作意図・設計方針

このアプリは、
「やることを考えること自体が負担になる」状態を減らすことを目的に設計しました。

* **今日やるタスクは最大3件**
  → 優先順位を強制的に意識できる
* **1タスクあたり最大30分**
  → タスクを細分化し、着手しやすくする
* **ドラッグ操作中心のUI**
  → 考えずに直感的に整理できる

タスク管理を「管理」ではなく
**行動を後押しする仕組み**として設計しています。

## 実装上の工夫

- **フロントエンドとバックエンドの分離**  
  React と Node.js を分離し、REST APIで連携しています。

- **型共有による安全なデータ通信**  
  フロントとバックエンドで型を共有し、API通信の不整合を防いでいます。

- **日付切り替え時の自動移動ロジック**  
  日付変更時にタスクを「明日やる」から「今日やる」へ自動移動します。タスクの優先度を維持し、継続的に行動できるようにしています。

- **ドラッグ＆ドロップとデータ連携**  
  UI操作での変更をAPI経由でDBに反映しています。
