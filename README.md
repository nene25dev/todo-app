# タスク管理アプリ

考えずに行動に移すための、シンプルなタスク管理アプリです。
タスクを「思いつき / 今日やる / 明日やる」に分けることで、
**やることを溜めすぎず、今やるべきことに集中できる設計**にしています。

※ 初回起動時は、動作確認用のサンプルタスクが表示されます。

---

## デモ

[https://nene25dev-todo-app.netlify.app/](https://nene25dev-todo-app.netlify.app/)

---

## 主な機能

* タスクの追加 / 編集 / 削除 / 復元
* タスクの分類（思いつき / 今日やる / 明日やる）
* ドラッグ＆ドロップによるタスク移動
* 完了したタスク・ごみ箱の管理
* フィルター切り替え（全て / 完了 / 削除）
* localForage によるブラウザローカル永続化

---

## 制作意図・設計方針

このアプリは、
「やることを考えること自体が負担になる」状態を減らすことを目的に設計しました。

* **今日やるタスクは最大3件まで**
  → 優先順位を強制的に意識できる
* **1タスクあたり最大30分**
  → タスクを細分化し、着手しやすくする
* **ドラッグ操作中心のUI**
  → 考えずに直感的に整理できる

タスク管理を「管理」ではなく
**行動を後押しする仕組み**として捉えています。

## 使用技術

* React
* TypeScript
* Vite
* localForage（IndexedDB / localStorage ラッパー）

---

## 工夫した点・こだわり

* 状態管理は React Hooks を使用
* 初回起動時のみサンプルデータを投入し、以降はユーザー操作を優先
* タスク移動処理は共通ロジックに集約し、並び替えと分類変更を同時に扱える設計
