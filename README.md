
# タスク管理アプリ
※ 初回起動時は動作確認用のサンプルタスクが表示されます。

## 概要
タスクを「思いつき / 今日やる / 明日やる」に分けて管理できる、優先順位がつけやすいタスク管理アプリです。
「考えずに行動に移れる」ことを重視し、タスク過多になりやすい人向けに設計しました。

## デモ
https://（デプロイURL）

## 使用技術
- React
- TypeScript
- Vite
- localForage（ブラウザローカル永続化）

## 主な機能
- タスクの追加 / 編集 / 削除 / 復元
- タスクの分別「思いつき」「今日やる」「明日やる」
- タスクのドラッグ＆ドロップ移動
- タスクの完了 / ゴミ箱管理
- フィルター切り替え（全て / 完了 / 削除）
- localForage によるデータ永続化

## 工夫した点・こだわり
- タスクが増えすぎないように、「今日やる」タスクは最大3件までに制限
- タスクを細分化しやすくするため、1タスクあたりの所要時間を最大30分に設定
- ドラッグ操作で直感的にタスクを整理できるUIを実装
